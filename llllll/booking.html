Help Page

How to Use This Website

  • Home: Overview of the site layout and navigation.
  • Gallery & Data: View images and a data table. Click images for details. Table is styled for readability.
  • Booking: Fill out the guest booking form. All fields are required. Select your major, gender, hobbies, and add comments. Use the submit button to send your booking.
  • Help: You're here! Find instructions for using each page and interacting with forms, images, and tables.
  • About: Learn about the project and the author.

Instructions

  • Use the navigation bar at the top to switch between pages.
  • On the Gallery page, hover over images for effects and view the data table for information.
  • On the Booking page, complete all fields before submitting. Required fields are marked and validated.
  • For any questions, refer to this Help page or contact the author via the About page.
